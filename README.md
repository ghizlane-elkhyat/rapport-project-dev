# 🎓 Générateur Automatique de Rapports Parents

## 📌 Description

Le projet consiste à développer une application web permettant de générer automatiquement des rapports pédagogiques destinés aux parents.

L’objectif principal est de :

- Centraliser les informations des élèves
- Faciliter la saisie des données pédagogiques
- Générer automatiquement des rapports PDF
- Envoyer les rapports par email
- Conserver un historique des rapports

---

## 🚀 Fonctionnalités

- 📚 Gestion des formations et des étudiants
- 📝 Création de rapports pédagogiques
- 📊 Ajout d’appréciations personnalisées
- 📄 Génération automatique de PDF (DomPDF)
- 📧 Envoi des rapports par email (SMTP)
- 🕓 Historique des rapports

---

## 🛠️ Technologies utilisées

### 🎨 Frontend
